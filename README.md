## 작품 주제
Java를 가지고 간단한 리듬게임을 연습삼아 만들어 보았습니다.<br>

곡에 맞추어 노트가 떨어지고 기준 라인에 내려왔을 때 히트시켜서 점수를 얻는 게임입니다.  난이도는 1부터 5까지이고, 총 4곡에 대한 리듬게임이 있습니다.
게임이 끝난 후 얻은 점수에 대해서 등급을 매겨 화면에 보여줍니다.
