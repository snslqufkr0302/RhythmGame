# 개인 프로젝트
리듬게임 ( RhythmGame )
<br>
<br>



## 작품 주제
Java를 가지고 cm송을 주제로 하여 리듬게임을 만들어보았습니다.<br>

곡에 맞추어 노트가 떨어지고 기준 라인에 내려왔을 때 키보드의 S, D, F, J, K, L, Space bar 를 눌러 히트시켜서 점수를 얻는 게임입니다.  
난이도는 1부터 5까지이고, 총 2곡에 대한 리듬게임이 있습니다.
게임이 끝난 후 얻은 점수에 대해서 등급 S, A, B, C 로 나누어 화면에 보여줍니다.
